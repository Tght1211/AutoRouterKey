# 📧 Outlook 账号管理器

一个现代化的 Web 界面，用于管理 Outlook 注册程序生成的账号密码，支持快捷复制、状态标记和 OpenRouter 注册管理。

## ✨ 主要功能

### 🎯 核心功能
- **一键复制**: 快速复制邮箱地址和密码到剪贴板
- **状态管理**: 标记账号状态（可用/已注册OpenRouter/已使用）
- **OpenRouter追踪**: 专门的开关来跟踪OpenRouter注册状态
- **实时搜索**: 快速搜索和过滤账号
- **数据导出**: 支持导出账号数据为文本文件
- **响应式设计**: 完美适配桌面和移动设备

### 📊 统计功能
- 总账号数统计
- 可用账号计数
- OpenRouter注册状态统计
- 已使用账号跟踪

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

```bash
# 直接运行启动脚本
./start_manager.sh
```

启动脚本会自动：
- 检查并激活虚拟环境
- 安装缺失的依赖
- 创建必要的目录和示例文件
- 启动Web服务器

### 方法二：手动启动

```bash
# 1. 激活虚拟环境
source venv/bin/activate

# 2. 安装Web服务器依赖
pip install flask flask-cors

# 3. 启动服务器
python3 main.py web
```

### 访问管理界面

启动成功后，在浏览器中访问：
- **主界面**: http://localhost:5010
- **API文档**: http://localhost:5010/api/accounts

## 📋 界面功能详解

### 1. 顶部统计区域
显示账号总数、可用数量、OpenRouter注册数量和已使用数量的实时统计。

### 2. 控制栏
- **🔄 刷新账号**: 重新读取最新的账号文件
- **➕ 添加账号**: 手动添加新的账号信息
- **📤 导出账号**: 导出所有账号数据为文本文件
- **🔍 搜索框**: 实时搜索账号或密码

### 3. 账号表格
每行包含：
- **序号**: 账号排序编号
- **邮箱账号**: 显示完整邮箱地址，点击📋可复制
- **密码**: 显示密码，点击📋可复制
- **状态下拉**: 可选择"可用"、"已注册OpenRouter"、"已使用"
- **OpenRouter开关**: 独立的注册状态切换
- **创建时间**: 账号创建或导入时间
- **操作按钮**: 编辑和删除功能

## 🔧 配置和数据

### 文件结构
```
outlookregister/
├── src/
│   ├── web/
│   │   ├── server.py         # Python后端服务器 (原 account_server.py)
│   │   └── templates/
│   │       └── manager.html  # Web界面文件 (原 account_manager.html)
├── scripts/
│   └── start_manager.sh      # 启动脚本
├── config/
│   └── app.json              # 配置文件
├── data/
│   ├── accounts.json         # 主要账号数据（包含状态、元数据等）
│   └── results/              # 临时结果目录
└── 账号管理器使用说明.md     # 本文档
```

### 数据同步
- **实时读取**: 自动读取 `data/accounts.json` 中的账号数据
- **状态持久化**: 账号状态和OpenRouter标记保存在 `accounts_data.json`
- **双向同步**: 支持Web界面和文件系统的双向数据同步

## 📱 使用技巧

### 快捷操作
1. **快速复制**: 点击邮箱或密码旁的📋图标
2. **批量标记**: 使用状态下拉菜单快速标记多个账号
3. **搜索过滤**: 在搜索框输入关键词快速定位账号
4. **导出备份**: 定期导出账号数据作为备份

### 状态管理建议
- **可用**: 新注册且未使用的账号
- **已注册OpenRouter**: 已在OpenRouter平台注册的账号
- **已使用**: 已被其他用途使用的账号

### OpenRouter管理
- 使用专门的开关来跟踪OpenRouter注册状态
- 独立于通用状态管理，可以更精确地跟踪使用情况

## 🔌 API接口

管理器提供RESTful API，可以与其他工具集成：

- `GET /api/accounts` - 获取所有账号
- `PUT /api/accounts/{id}/status` - 更新账号状态
- `PUT /api/accounts/{id}/openrouter` - 更新OpenRouter状态
- `GET /api/accounts/stats` - 获取统计信息
- `GET /api/accounts/export` - 导出账号数据

## 🛠️ 故障排除

### 常见问题

**Q: 页面显示"连接服务器失败"**
A: 检查Python服务器是否正常启动，确认端口5010未被占用

**Q: 账号数据不显示**
A: 确认 `data/accounts.json` 文件存在且格式正确

**Q: 状态更新不生效**
A: 检查 `data/accounts.json` 文件权限，确保可写

**Q: 启动脚本执行失败**
A: 确认脚本有执行权限：`chmod +x start_manager.sh`

### 文件格式要求

账号文件格式（每行一个账号）：
```
email@outlook.com: password123
another@outlook.com: pass456
```
